# Frontrend 💖  
今どきのコード、まるっと分析するポップなトレンド解析ツール！



---

## 💡 Frontrendとは？
「最近のLPって、どんなマークアップ？」「CSS設計の流行は？」  
そんな疑問をまるっと解決してくれる、**ソースコード分析SaaS**です！

- ギャラリーサイトのURLをペタッと貼るだけ
- LP一覧のURLを自動で抽出
- マークアップやCSSの傾向を分析・統計化
- ダッシュボードでポップに可視化💘
